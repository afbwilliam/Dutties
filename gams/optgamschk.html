Summary of GAMSCHK Options

General GAMSCHK options

CNS solver for CNS problems
DESCALE controls treatment of scaling
DNLP solver for DNLP problems
EQUBLOCK maximum number of equation blocks allowed
LEVELFILT numerical value of exponent on "unbounded levels"
LP solver for LP problems
MARGFILT numerical value of exponent on "infeasible marginals"
MCP solver for MCP problems
MINLP solver for MINLP problems
MIP solver for MIP problems
MPEC solver for MPEC problems
NLP solver for NLP problems
NOSOLVE suppresses solution of the problem
OPTFILE solver options file number
RMINLP solver for RMINLP problems
RMIP solver for RMIP problems
RMPEC solver for RMPEC problems
SOLVE forces solution of the problem
SOLVERNAME solver for any problems
VARBLOCK maximum number of variable blocks allowed

Detailed Descriptions of GAMSCHK Options

CNS (string) solver for CNS problems

DESCALE (string) controls treatment of scaling


(default = part)

DNLP (string) solver for DNLP problems

EQUBLOCK (integer) maximum number of equation blocks allowed


(default = -5)

LEVELFILT (integer) numerical value of exponent on "unbounded levels"


(default = 6)

LP (string) solver for LP problems

MARGFILT (integer) numerical value of exponent on "infeasible marginals"


(default = 6)

MCP (string) solver for MCP problems

MINLP (string) solver for MINLP problems

MIP (string) solver for MIP problems

MPEC (string) solver for MPEC problems

NLP (string) solver for NLP problems

NOSOLVE (string) suppresses solution of the problem

OPTFILE (string) solver options file number

RMINLP (string) solver for RMINLP problems

RMIP (string) solver for RMIP problems

RMPEC (string) solver for RMPEC problems

SOLVE (string) forces solution of the problem

SOLVERNAME (string) solver for any problems

VARBLOCK (integer) maximum number of variable blocks allowed


(default = -5)
