Detailed Descriptions of LGO Options

acc_tr (real) Global search termination - acceptability threshold

Global search termination criterion parameter (acceptability threshold). The global search phase (BB, GARS, or MS) ends, if an overall merit function value is found in the global search phase that is not greater than acc_tr.
(default = -1e10)

bad_obj (real) Default value for objective function, if evaluation errors occur


(default = 1e8)

CallConopt (real) call CONOPT to get duals for final point

Number of seconds given for cleanup phase using CONOPT. CONOPT terminates after at most CallConopt seconds. The cleanup phase determines duals for final solution point.
(default = 5)

con_tol (real) Maximal constraint violation tolerance in local search


(default = 1e-6)

debug (integer) Debug option.

Prints out complete LGO status report to listing file.
(default = 0)

fct_trg (real) Partial stopping criterion in second local search phase

Target objective function value.
(default = -1e10)

fi_tol (real) Local search (merit function improvement) tolerance


(default = 1e-6)

g_maxfct (integer) Maximum number of function evaluations in global search

Maximum number of merit (model) function evaluations before termination of global search phase (BB, GARS, or MS). In the default setting, n is the number of variables and m is the number of constraints. The difficulty of global optimization models varies greatly: for difficult models, g_maxfct can be increased as deemed necessary.
(default = -1)

Include (string) Start reading from a new file

irngs (integer) Random number seed


(default = 0)

kt_tol (real) Kuhn-Tucker local optimality condition violation tolerance


(default = 1e-6)

log_err (integer) Log first log_err errors

Error reported (if applicable) every log_err iterations.
(default = 10)

log_iter (integer) Iteration log time interval.

Log output occurs every log_iter iterations.
(default = 10)

log_time (real) Iteration log time interval in seconds.

Log output occurs every log_time seconds.
(default = 0.5)

max_nosuc (integer) Maximum number of function evaluations without improvement

Maximum number of merit function evaluations in global search phase (BB, GARS, or MS) where no improvement is made. Algorithm phase terminates upon reaching this limit. The default of -1 uses 100(nvars+ncons), where nvars is the number of variables and ncons the number of constraints.
(default = -1)

opmode (integer) Operational mode


(default = 3)
0LS. Local search from the given nominal solution without a preceding local search (LS)
1BB + LS. Global branch-and-bound search and local search (BB+LS)
2RS+LS. Global adaptive random search and local search (GARS+LS)
3MS+LS. Global multistart random search and local search (MS+LS)

penmult (real) Constraint penalty multiplier.

Global merit function is defined as objective + the constraint violations weighted by penmult.
(default = 100)

tlimit (real) Time limit in seconds.
   Synonym: reslim

This is equivalent to the GAMS option reslim. If specified, this overrides the GAMS reslim option.
(default = 1000)

var_lo (real) Smallest (default) lower bound, unless set by user.


(default = -1e6)

var_up (real) Largest (default) upper bound, unless set by user.


(default = 1e6)
