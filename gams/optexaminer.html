EXAMINER Options

For more information about this solver please inspect the complete EXAMINER manual.

Summary of EXAMINER Options

General Options

absXScale make scale factors dependent on x values
checkDump dump file for results of Examiner checks
dumpGamsPoint dump the GamsPoint to a basis file in GAMS source format
dumpInitPoint dump the InitPoint to a basis file in GAMS source format
dumpSoluPoint dump the SoluPoint to a basis file in GAMS source format
dumpSolvPoint dump the SolvPoint to a basis file in GAMS source format
examineGamsPoint examine the GamsPoint
examineInitPoint examine the InitPoint
examineSoluPoint examine the SoluPoint
examineSolvPoint examine the SolvPoint
fCheckAll forces all the checks on or off
fCheckATTR forces the model attributes check
fCheckDCMP forces the dual complementary slackness check
fCheckDCON forces the dual contraint feasibility check
fCheckDVAR forces the dual variable feasibility check
fCheckPCMP forces the primal complementary slackness check
fCheckPCON forces the primal contraint feasibility check
fCheckPVAR forces the primal variable feasibility check
GDXDump not yet implemented
perpSys print FOC for visual inspection
returnGamsPoint return the GamsPoint as a solution to GAMS/Base
returnInitPoint return the InitPoint as a solution to GAMS/Base
returnSoluPoint return the SoluPoint as a solution to GAMS/Base
returnSolvPoint return the SolvPoint as a solution to GAMS/Base
scaled apply checks to a scaled version of the model
scaleLB lower bound for applied row scales
scaleUB upper bound for applied row scales
showIndex show row/column indices in addition to their names
showSlacks explicitly show the infeasibilities or slacks for failed checks
solutionFile read solution from pre-existing solution file
subSolver subsolver to run
subSolverOpt optfile value to pass to the subsolver
trace Examiner trace file
unScaled apply checks to the original, unscaled version of the model

Tolerance Options

dualCSTol dual complementary slackness tolerance, i.e. between the dual variables and the primal constraints
dualFeasTol tolerance used to check feasibility of the dual variables and constraints
ECTol equilibrium condition complementarity tolerance
primalCSTol primal complementary slackness tolerance, i.e. between the primal variables and the dual constraints
primalFeasTol tolerance used to check feasibility of the primal variables and constraints
showTol relative tolerance for showSlacks - we only want to see explicit slacks that are relatively small

Detailed Descriptions of EXAMINER Options

absXScale (integer) make scale factors dependent on x values


(default = 1)

checkDump (string) dump file for results of Examiner checks

dualCSTol (real) dual complementary slackness tolerance, i.e. between the dual variables and the primal constraints


(default = 1e-007)

dualFeasTol (real) tolerance used to check feasibility of the dual variables and constraints


(default = 1e-006)

dumpGamsPoint (integer) dump the GamsPoint to a basis file in GAMS source format


(default = 0)

dumpInitPoint (integer) dump the InitPoint to a basis file in GAMS source format


(default = 0)

dumpSoluPoint (integer) dump the SoluPoint to a basis file in GAMS source format


(default = 0)

dumpSolvPoint (integer) dump the SolvPoint to a basis file in GAMS source format


(default = 0)

ECTol (real) equilibrium condition complementarity tolerance


(default = 1e-006)

examineGamsPoint (integer) examine the GamsPoint


(default = 0)

examineInitPoint (integer) examine the InitPoint


(default = 0)

examineSoluPoint (integer) examine the SoluPoint


(default = 0)

examineSolvPoint (integer) examine the SolvPoint


(default = 0)

fCheckAll (integer) forces all the checks on or off


(default = 0)

fCheckATTR (integer) forces the model attributes check


(default = 0)

fCheckDCMP (integer) forces the dual complementary slackness check


(default = 0)

fCheckDCON (integer) forces the dual contraint feasibility check


(default = 0)

fCheckDVAR (integer) forces the dual variable feasibility check


(default = 0)

fCheckPCMP (integer) forces the primal complementary slackness check


(default = 0)

fCheckPCON (integer) forces the primal contraint feasibility check


(default = 0)

fCheckPVAR (integer) forces the primal variable feasibility check


(default = 0)

GDXDump (string) not yet implemented

perpSys (integer) print FOC for visual inspection


(default = 0)

primalCSTol (real) primal complementary slackness tolerance, i.e. between the primal variables and the dual constraints


(default = 1e-007)

primalFeasTol (real) tolerance used to check feasibility of the primal variables and constraints


(default = 1e-006)

returnGamsPoint (integer) return the GamsPoint as a solution to GAMS/Base


(default = 0)

returnInitPoint (integer) return the InitPoint as a solution to GAMS/Base


(default = 0)

returnSoluPoint (integer) return the SoluPoint as a solution to GAMS/Base


(default = 0)

returnSolvPoint (integer) return the SolvPoint as a solution to GAMS/Base


(default = 0)

scaled (integer) apply checks to a scaled version of the model


(default = 0)

scaleLB (real) lower bound for applied row scales


(default = 1)

scaleUB (real) upper bound for applied row scales


(default = maxdouble)

showIndex (integer) show row/column indices in addition to their names


(default = 0)

showSlacks (integer) explicitly show the infeasibilities or slacks for failed checks


(default = 0)

showTol (real) relative tolerance for showSlacks - we only want to see explicit slacks that are relatively small


(default = 1e-004)

solutionFile (string) read solution from pre-existing solution file

subSolver (string) subsolver to run

subSolverOpt (integer) optfile value to pass to the subsolver


(default = 0)

trace (string) Examiner trace file

unScaled (integer) apply checks to the original, unscaled version of the model


(default = 1)
