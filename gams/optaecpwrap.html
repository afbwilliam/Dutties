Detailed Descriptions of AECPWRAP Options

AlphaStr (integer) Alpha_Strategy


(default = 1)
0No alpha updating used on the cutting planes
1Alpha updating (should be used if termination criterion>1)

Beta (real) Beta


(default = 1.30)

CallConopt (integer) Call conopt to get duals


(default = 0)
0Conopt is not called
1Automatically call Conopt to obtain dual values for the solution point returned

C_BranchDirect (integer) Branching_direction


(default = 0)

C_MaxCpuTime (integer) Tree_memory_limit


(default = 1000)

C_MipStrBackT (real) MIP_strategy_backtrack


(default = 0.9999)

C_MipTolIntegral (real) MIP_tolerance_integrality


(default = 1E-5)

C_NodeSelectStr (integer) Node_selection_strategy


(default = 1)

C_Presolve (integer) Preprocessing_presolve


(default = 0)

C_RelMipGapTol (real) Relative_MIPgap_tolerance


(default = 0.0001)

C_SubLpAlg (integer) Subproblem_LP_algorithm


(default = 0)

C_VarSelectStr (integer) Variable_selection_strategy


(default = 0)

DefaultSettings (integer) 1=Convex, 2=Pseudo-convex constraints, 3=Pseudo-convex


(default = 2)
0EcpStr=1, EcpLinStr=4, EcpTermCrit=1,EcpC_MipSolLim1=1
1EcpStr=1, EcpLinStr=1, EcpTermCrit=1,EcpC_MipSolLim1=1
2EcpStr=2, EcpLinStr=1, EcpTermCrit=3,EcpC_MipSolLim1=1
3EcpStr=3, EcpLinStr=1, EcpTermCrit=4,EcpC_MipSolLim1=1

EcpC_MipSolLim1 (integer) MIP_solutions_limit_(initial)


(default = 1)

EcpIterLim (integer) ECP_Iteration_Limit


(default = GAMS IterLim)

EcpLinStr (integer) ECP_Linearization_Strategy


(default = 1)
0Add cutting planes only for max g_i >= epsilon_g
1Add cutting planes for 25% of max g_i >= epsilon_g
2Add cutting planes for 50% of max g_i >= epsilon_g
3Add cutting planes for 75% of max g_i >= epsilon_g
4Add cutting planes for all g_i >= epsilon_g

EcpStartStr (integer) ECP_Starting_Strategy


(default = 1)
1First ECP Iteration solved as a MIP problem
2First ECP Iteration solved as a Relaxed MIP problem

EcpStr (integer) Change only strategy 1,2 or 3


(default = 2)
1No Alpha updating on cutting planes
2Alpha updating on cutting planes
3Strategy 2 + objective function reduction constraint

EcpTermCrit (integer) ECPTermination_Criterion


(default = 3)
1Max g_i <= epsilon_g
2Criterion 1 + Special criterion for Quasi-Convex NLP
3Criterion 1 + Alpha >= g_k / epsilon_h
4Criterion 3 + abs(f(z)-mu) <= epsilon_f

EpsilonF (real) Epsilon_f


(default = 0.001)

EpsilonG (real) Epsilon_g


(default = 0.001)

EpsilonHStr (integer) Epsilon_h_Strategy


(default = 1)
0Epsilon_h = Epsilon_z
1Epsilon_h = Epsilon_z * Norm of gradient at k

EpsilonZ (real) Epsilon_z


(default = 0.1)

Gamma (real) Gamma


(default = 2)

InfBnd (real) Infinity value to be used on bounds


(default = 1E10)

MaxCpuTime (real) Maximum CPU-time


(default = GAMS ResLim)

ModAlphaFiles (integer) Allow_Modified_Alpha_Files


(default = 0)
0Problem in *.ecp file generates alpha2.for, alpha3.for automatically
1As 0, but STOP before compiling Alpha-files (implicit MINLP models)

PcoStr (integer) PCO_Strategy


(default = 3)
1Remove old objective function cuts when obj.fun. is improved
2Replace old objective function cuts when obj.fun. is improved
3Remove old objective function cuts when obj.fun. is improved + line search to the best level curve
4Replace old objective function cuts when obj.fun. is improved + line search to the best level curve

ResMemSize (integer) Reserved_Memory_Size


(default = 5000000)
