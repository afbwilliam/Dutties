JAMS Options

All options available through GAMS/JAMS are described in this document.

For more information about this solver please inspect the complete JAMS manual.

Detailed Descriptions of JAMS Options

Dict (string) Creates dictionary file

Generate a dictionary file that maps original variable and equation names to the ones of the scalar JAMS model.
(default = dict.txt)

DisjBinRelax (integer) Relax requirement that disjunction variables are binary


(default = 0)

EMPInfoFile (string) Path and name of file containing additional EMP information

Path and name of file containing additional EMP information as disjunctions, perpendiculars, bilevel characterization etc. By default, the file is expected to exist in the GAMS scratch directory of the current run (gams.scrdir). If this option is used, EMP searches for the file based on the provided relative or absolute path. In case of a given relative path, it is assumed to be relative to the working directory.

FileName (string) Filename of generated scalar GAMS model

The generated reformulated model will be written into a file with this name
(default = emp.gms)

KeepObj (integer) Keep original objective function and variables in generates MCP model

This option is only valid for reformulations into complementarity systems. If KeepObj is set the generated MCP program will incorporate the original objective function and the Karush-Kuhn-Tucker condition of the original objective variable.
(default = 0)

MargTol (real) Only report marginals with an absolute value above the tolerance

The user can define a tolerance below which the marginal values will not be reported.
(default = 1e-6)

ObjVarName (string) Name of objective variable

Name of objective variable in generated model
(default = objvar)

SubSolvePar (string) User defined GAMS parameters for subsolve

The user can define GAMS parameters for the subsolve.

SubSolver (string) Subsolver to run

Subsolver used to solve the reformulated model. The subsolver chosen has to be suitable for the reformulation type chosen. The user can also provide a solver related option file, see SubSolverOpt.

SubSolverOpt (integer) Optfile value to pass to the subsolver

Number of option file for subsolver.
(default = 1)

Terminate (integer) Generate the GAMS source code of the reformulated Model

Generate the GAMS source code of the reformulated Model in a file with name FileName and terminate without solving the model.
(default = 0)
