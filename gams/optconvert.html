Detailed Descriptions of CONVERT Options

All (integer) Generates all supported file formats

AlphaECP (string) Generates AlphaECP input file


(default = alpha.ecp)

Ampl (string) Generates Ampl input file


(default = ampl.mod)

AmplNLC (string) Generate Ampl NLC compatible file


(default = amplnlc.c)

Analyze (string) Generates three text files for rows columns and matrix


(default = analyze.txt)

AnalyzeS (string) Generates short form of Analyze


(default = analyzes.txt)

Baron (string) Generates Baron input file


(default = gams.bar)

ConeReform (integer) Reformulation of cone =C= constraints to NLP format


(default = 0)
0keep =C= format
1convert conic constraints to NLP format

CplexLP (string) Generate CPLEX LP format input file


(default = cplex.lp)

CplexMPS (string) Generate CPLEX MPS format input file


(default = cplex.mps)

CppAD (string) Generate AD IPOPT format


(default = cppadFunc.cpps)

Dict (string) Generate Convert to GAMS Dictionary


(default = dict.txt)

DictMap (string) Generate Convert to GAMS Dictionary Map


(default = dictmap.gdx)

FileList (string) Generate file list of file formats generated


(default = files.txt)

FixedMPS (string) Generate fixed format MPS file


(default = fixed.mps)

Gams (string) Generate GAMS scalar model. This is the default conversion format used.


(default = gams.gms)

GmsInsert (string) Insert text before solve


(default = $if NOT '%gams.u1%' == '' $include '%gams.u1%')

Hessian (string) Writes GDX version of current point - Jacobian and Hessian


(default = hessian.gdx)

Include (string) Start reading from a new file

Jacobian (string) Writes GDX version of current point - Jacobian


(default = jacobian.gdx)

Lago (string) Generate a partial Lago file


(default = lago.gms)

Lgo (string) Generate an LGO Fortran file


(default = lgomain.for)

LindoMPI (string) Generate Lindo MPI file


(default = lindo.mpi)

Lingo (string) Generate Lingo input file


(default = lingo.lng)

LocalSolver (string) Generate LocalSolver input file (only with ConvertD)


(default = localsolver.lsp)

LSPSol (string) Generate Output function in LocalSolver input file (only with ConvertD)


(default = lspsol.gms)

Match (integer) Force a complete match for all MCP variable/equation pairs

Memo (string) Generate a memo file containing model statistics and files created.


(default = memo.txt)

Minopt (string) Generate Minopt input file


(default = minopt.dat)

NLP2MCP (string) Generates GAMS scalar MCP model


(default = gamsmcp.gms)

ObjVar (string) Name of objective variable.


(default = GAMS index name, e.g. x1)

OSiL (string) Generates Optimization Services instance Language (OSil) file


(default = osil.xml)

Reform (integer) Force reformulations


(default = 100)

SFS (string) Generates Solver Foundation Services OML file


(default = sfs.oml)

Terminate (integer) Force Gams to terminate after conversion

ViennaDag (string) Generate Vienna Dag input file


(default = vienna.dag)
